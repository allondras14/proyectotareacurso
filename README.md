# 🎲 Juego del Número Secreto

¡Bienvenido, valiente debugger de la suerte!  
Este es un juego web tan sencillo que hasta tu compilador lo entendería… pero no te confíes: tu misión es adivinar el **Número Secreto™** generado aleatoriamente dentro de un rango específico.  

El objetivo es simple: encontrar el número antes de que tu paciencia haga `throw new RageQuitException();`.

---

## ✨ Características Principales

*   **Interfaz Minimalista:** Tan limpia que hasta `lint` se queda sin trabajo.
*   **Feedback Instantáneo:** El juego te dice si tu número es **mayor** o **menor**… como un `if/else` con mucho estilo.
*   **Conteo de Intentos:** Porque todo héroe necesita saber cuántos `while(true)` le tomó llegar a la gloria.
*   **Botón para Reiniciar:** ¿Fallaste? No pasa nada, dale al botón y haz un `reset()` a tu dignidad.
*   **Diseño Responsive:** Juega desde tu PC, tablet o celular… ¡incluso desde el baño si quieres, no juzgamos!

---

## 🛠️ Tecnologías Utilizadas

Este proyecto fue forjado con las armas clásicas del front-end:

*   **HTML5:** La estructura, el esqueleto, el `div` que sostiene todo.
*   **CSS3:** El maquillaje, las animaciones y el glamour.
*   **JavaScript:** El cerebro detrás del caos, el que decide si ríes o lloras.
